# JPA Bidirectional One-to-One Mapping

Demonstrates bidirectional one-to-one relationships in JPA.

## Features

- Bidirectional navigation between entities
- @OneToOne with mappedBy
- Cascade operations
- Proper relationship management

## Key Concepts

- Owner side vs inverse side
- mappedBy attribute
- Bidirectional synchronization

## Run

```bash
mvn clean install
```
