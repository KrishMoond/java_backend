# Hibernate Basics

JPA/Hibernate fundamentals with entity operations.

## Entities

- **Student**: Student with percentage and DOB
- **Employee**: Employee with salary

## Classes

- **Demo**: Insert student records
- **Demofetch**: Fetch and delete operations
- **DemoFind**: Find and update operations

## Features

- EntityManager operations
- persist(), find(), merge(), remove()
- Transaction management
- Entity lifecycle

## Run

```bash
mvn exec:java -Dexec.mainClass="com.prac.Demo"
```
